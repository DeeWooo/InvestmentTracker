# 我为什么开始记录每笔交易

> 从亏损 18% 到稳定盈利，我只做了一件事：开始记录。

---

## 一、我曾经是典型的散户

2018 年初，我开始炒股。

当时的状态是这样的：
- 看到朋友说某只股票涨了，马上买入
- 看到新闻说某个板块很热，冲动加仓
- 看到股价下跌 5%，恐慌性割肉
- 看到股价上涨 3%，贪心地想等它涨到 10%

三个月后，我打开券商 APP，惊讶地发现：**账户亏损 50%**。

更可怕的是，我完全说不出自己为什么亏钱。

---

## 二、转折点：一次痛苦的复盘

有一天，我试图复盘自己的交易。

我打开券商 APP 的"历史成交"，看到密密麻麻的交易记录：
```
2018-01-31  买入  格林美  1000股  @ 6745
2018-01-31  卖出  海螺水泥  200股  @ 6826.2 
2018-02-01  买入  向日葵 100股 @ 345
2018-02-28  买入  五矿资本 600股 @ 6491
2018-03-02  卖出  福田汽车 1500股 @ 3816.09
2018-03-06  买入  西藏天路 1000股 @ 9175
2018-03-07  卖出  西藏天路 900股 @ 8131.7
...
```

我盯着这些数字看了半小时，脑子一片混乱：
- **为什么买入？** 不记得了。
- **为什么卖出？** 不记得了。
- **卖出代表盈利还是亏损？** 看不出来
- **当时的心理？** 不记得了。
- **这些交易有什么规律？** 看不出来。

那一刻我意识到：**我不是在投资，我是在赌博。**

---

## 三、开始记录：建立我的投资日志

从那天起，我做了一个决定：**记录每一笔交易**。

我开始用 Excel 表格记录：

| 日期 | 代码 | 名称 | 买入价 | 数量 | 原因 | 预期止盈 | 预期止损 |
|------|------|------|--------|------|------|---------|---------|
| 2018-05-04 | 600863 | 内蒙华电 | 2.6700 | 100 | 大国企改革 | 2.937 (+10%) | 2.403 (-10%) |
| 2018-05-07 | 600166 | 福田汽车 | 2.29 | 300 | 新能源政策利好 | 2.519 (+10%) | 2.061 (-10%) |


这个表格很简单,但它迫使我在**买入之前思考**：
- ✅ 我为什么要买这只股票？
- ✅ 我预期它涨到多少？
- ✅ 如果跌了,我什么时候止损？

---

## 四、三个月后：数据告诉我真相

三个月后，我的投资日志有了 20 条记录。

我开始分析这些数据，发现了惊人的规律：

### 发现 1：我的失败交易都有共同特征
```
失败交易的共同点：
- 80% 是因为"听朋友说"或"看新闻"
- 90% 没有设定止损点
- 100% 在下跌时没有按计划止损
```

### 发现 2：我的成功交易也有共同特征
```
成功交易的共同点：
- 都是自己研究后的决定
- 都有明确的止盈/止损点
- 都按计划执行了
```

### 发现 3：我的真实胜率
```
总交易次数：20 笔
盈利次数：8 笔
亏损次数：12 笔
胜率：40%

但是！
平均盈利：+12%
平均亏损：-5%（因为设了止损）
总体收益：+3.2%
```

**这些数据告诉我**：我不需要提高胜率，我只需要**严格执行止损规则**。

---

## 五、建立纪律：从记录到改进

有了数据，我开始建立自己的交易纪律：

### 纪律 1：买入前必须回答三个问题
1. 为什么买这只股票？（必须写下理由）
2. 预期涨到多少卖出？（+10%、+20%）
3. 跌到多少止损？（-10%）

### 纪律 2：禁止追涨杀跌
```
见涨不急 → 等三天观察
见跌不慌 → 查看是否触发止损点
```

### 纪律 3：每月复盘
```
每月 1 号，查看上月所有交易
分析：哪些做对了？哪些做错了？
调整：下月的改进点是什么？
```

## 六、一年后：从亏损到盈利

一年后，我再次打开我的投资日志。

数据对比让我震惊：

| 时间 | 总投入 | 总收益 | 收益率 | 胜率 | 备注 |
|------|--------|--------|--------|------|------|
| 2018 Q1（记录前） | 2万 | -1万 | -50% | 不详 | 凭感觉交易 |
| 2018 Q2-Q4（记录后） | 2万 | +0.3万 | +15% | 45% | 有纪律交易 |
| 2019 全年 | 12万 | +1.9万 | +6% | 52% | 持续优化 |

**我没有突然变聪明，我只是有了纪律。** 哪怕在18年下半年的大熊市和19年的谷底。

## 七、为什么我开发了 InvestmentTracker

Excel 表格很好，但我发现了一些问题：
- ❌ 手动计算盈亏很麻烦
- ❌ 查看历史交易不方便
- ❌ 数据分析需要写公式

作为一名程序员，我决定开发一个工具来解决这些问题。

### InvestmentTracker 做了什么？

✅ **自动计算盈亏**
```
买入价：100 元
当前价：110 元
自动显示：+10%（盈利 10 元/股）
```

✅ **自动生成止盈止损点**
```
买入价：100 元
建议止盈：110 元（+10%）、120 元（+20%）
建议止损：90 元（-10%）
```

✅ **跨平台聚合**
```
富途账户：持仓 A、B、C
老虎账户：持仓 D、E
统一视图：A、B、C、D、E
```

✅ **数据完全本地化**
```
存储方式：本地 SQLite 数据库
隐私保护：数据不上传云端
离线可用：不需要联网
```

## 八、给新手的建议

如果你也是投资新手，我的建议是：

### 1. 立即开始记录
不需要复杂的工具，Excel 也可以。关键是**开始**。

必须记录的字段：
- 日期
- 代码/名称
- 买入价
- 数量
- 买入原因
- 预期止盈/止损点

### 2. 每笔交易前问自己
- 我为什么要买？
- 我预期涨到多少？
- 我能接受亏多少？

**如果回答不出来，就不要买。**

### 3. 定期复盘
- 每周看一次
- 每月总结一次
- 每季度调整策略

### 4. 相信数据，不相信直觉
```
直觉说：这只股票肯定会涨
数据说：你上次这么想，亏了 15%

听数据的。
```

## 十、InvestmentTracker 的诞生

我开发 InvestmentTracker 的初衷很简单：**让记录交易这件事变得更简单**。

### 目前已经实现的功能
- ✅ **持仓记录**：记录每笔买入（代码、价格、数量、日期、投资组合）
- ✅ **盈亏计算**：自动计算当前盈亏和盈亏比
- ✅ **投资组合视图**：按自定义组合分类管理持仓
- ✅ **本地存储**：SQLite 数据库，数据完全私密
- ✅ **平仓管理**：标记已平仓的持仓，保留历史记录
- ✅ **建议点位**：自动显示止盈止损参考价格

### 正在开发中的功能

- 🚧 交易统计报表（胜率、平均盈亏等）
